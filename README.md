# Jejuskin

Jejuskin est une boutique en ligne de produits de soins coréens, créée pour offrir les meilleurs soins de la peau à nos clients dans un style élégant et accessible. Notre site utilise une interface épurée et moderne pour améliorer l'expérience utilisateur lors de la découverte et de l'achat de produits de soins de la peau.

## Project Overview

Ce projet est conçu pour afficher une page d'accueil dynamique présentant divers produits de soins ainsi qu'une section de panier. Il a été construit en utilisant HTML, CSS et le framework Tailwind CSS, offrant une belle mise en page responsive ainsi qu'une navigation intuitive.

## Installation

Pour exécuter ce projet localement, suivez les étapes ci-dessous :

1. Clonez le dépôt : 
   ```bash
   git clone <lien_du_depot>
   ```
   
2. Accédez au répertoire du projet :
   ```bash
   cd jejuskin
   ```

3. Ouvrez le fichier `index.html` dans votre navigateur web.

## Usage

Une fois le projet ouvert dans votre navigateur, vous pouvez explorer les différentes sections de la boutique en ligne :
- Parcourez les produits disponibles dans la boutique.
- Ajoutez des articles à votre panier.
- Découvrez les étapes de la routine de soins coréens.

## Features

- **Interface Responsive :** Conçu pour s'adapter à tous les écrans (portables, tablettes et ordinateurs).
- **Navigation Fluide :** Accès facile aux différentes sections du site.
- **Panier Dynamique :** Affiche la quantité en temps réel et permet de mettre à jour ou valider les commandes.
- **Guides de produits :** Directives sur comment utiliser les produits de soins pour une routine de soins coréens parfaite.

## Dependencies

Le projet utilise les ressources suivantes :
- [Tailwind CSS](https://tailwindcss.com/) pour le style visuel.
- [Font Awesome](https://fontawesome.com/) pour les icônes.

Les dépendances sont intégrées via des CDN dans le fichier HTML. Aucun fichier `package.json` n'est présent dans le projet.

## Project Structure

Voici la structure des fichiers du projet :

```
jejuskin/
├── index.html          # La page d'accueil de la boutique en ligne
```

L'application est principalement constituée d'un unique fichier HTML qui contient tout le contenu nécessaire pour la boutique en ligne. Les styles sont incorporés via le framework Tailwind CSS chargé depuis un CDN, et les polices sont ajoutées depuis Google Fonts.
